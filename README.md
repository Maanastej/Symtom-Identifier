# Symptom Identifier

A modern web application for identifying symptoms and managing health records, built with React, Vite, and Supabase.

## Features

- **Symptom Tracking**: Log and identify symptoms for better health management.
- **Supabase Integration**: Secure data storage and authentication.
- **Responsive Design**: Optimized for both mobile and desktop views using Tailwind CSS.
- **Modern UI Components**: Built with shadcn/ui.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd symptom-identifier
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Set up environment variables:
   Create a `.env` file in the root directory with the following variables:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

5. Start the development server:
   ```sh
   npm run dev
   ```

- **Framework**: [React](https://reactjs.org/)
